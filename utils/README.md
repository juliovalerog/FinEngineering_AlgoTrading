# Utils folder

This folder now contains the reusable components that support the LBO teaching asset:

- `lbo_engine.py`: the shared analytical engine used by the notebook and the Streamlit demo.
- `gemini_reporting.py`: the optional Gemini-based reporting helper that turns model outputs into a short executive commentary.

The goal is to keep reusable logic in one place while preserving the notebook as the main didactic walkthrough.
